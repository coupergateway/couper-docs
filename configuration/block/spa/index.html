SPA

The spa blocks configure the Web serving for SPA assets. Can be defined multiple times as long as the base_path+paths is unique.

Block nameContextLabelNested block(s)
spaServer BlockOptionalCORS Block

bootstrap_filestring-Location of the bootstrap file.⚠ requiredbootstrap_file = "./htdocs/index.html"

disable_access_controltuple (string)-Disables access controls by name.-disable_access_control = ["foo"]
