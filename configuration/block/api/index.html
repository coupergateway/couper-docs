Api

The api block bundles endpoints under a certain base_path.

If an error occurred for api endpoints the response gets processed as JSON error with an error body payload. This can be customized via error_file.

Block nameContextLabelNested block(s)
apiServer BlockOptionalEndpoint Block(s), CORS Block, Error Handler Block(s)

Attribute allowed_methods

The default value * can be combined with additional methods. Methods are matched case-insensitively. Access-Control-Allow-Methods is only sent in response to a CORS preflight request, if the method requested by Access-Control-Request-Method is an allowed method.

Example: allowed_methods = ["GET", "POST"] or allowed_methods = ["*", "BREW"]
